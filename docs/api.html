Sidecar HTTP API

各 Lictor セッションは loopback (127.0.0.1) のみ で HTTP sidecar を listen する（port は子プロセスに LICTOR_PORT で渡る）。各行をクリックすると トグル展開し、パラメータ・動作・レスポンスを確認できる。

ベース URL http://127.0.0.1:${LICTOR_PORT}
